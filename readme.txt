Build Instructions:
1. Clone GitHub repository to a location of your choosing on your computer.
2. Open up a Terminal
3. Navigate to the place where the repo was cloned to
4. Because of the addition of a make file, all you need to do is type "make"
5. Clear up any errors/warnings that appear, there should be none.

Run Instructions:
1. Now that the file has built properly using the make file included, and all errors were cleared up, enter "./myshell" in the 
command line.
2. Now, you are inside the shell. The only commands that are supported thus far are ls, cd, grep, pwd, cat, and wc. Type exit to exit the shell.
